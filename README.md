# DiscordTerminal-JS
A Discord bot for sending command to linux ssh terminal and show output.

Usage:-
```
!t <bash command>
```

Example:-
```
!t lsb_release -a
```

Returns:-
```
Distributor ID:    Debian
Description:    Debian GNU/Linux 10 (buster)
Release:    10
Codename:    buster
```

## Deploy
Set config.json before running.

Install Modules:-
```
npm install
```

Run Program:-
```
node app.js
```
